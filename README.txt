Это творческое задание для интенсива от Академии Яндекса

1) скачайте проект в zip архив
2) разархивируйте скачанный файл
3) зайдите в powershell(shift+пкм) из папки newyandex
4) введите команду pip install -r requirements.txt
5) зайдите в powershell(shift+пкм) из папки website
6) введите команду py manage.py runserver
7) перейдите по ссылке http://127.0.0.1:8000/homepage/
8) после просмотра выключите терминал, прекращая работу
